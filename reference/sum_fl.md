# Summer Flounder

A dataset containing containing the number of fish caught at different
sites and timepoints.

## Usage

``` r
sum_fl
```

## Format

A `data.frame` with 350 rows and 13 columns:

- patch:

  Patch code

- lat_floor:

  Lowest latitute associated with the current patch

- year:

  Year

- y:

  Number of fish caught

- n_hauls:

  Number of hauls used to catch the fish

- stemp:

  Sea surface temperature

- ssalin:

  Sea surface salinity

- btemp:

  Sea bottom temperature

- bsalin:

  Sea bottom salinity

- lat:

  Avg. latitude

- lon:

  Avg. longitude

- depth:

  Avg. depth

- area_km2:

  Area in km^2^
