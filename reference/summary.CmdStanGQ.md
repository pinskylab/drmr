# Wrapper for the summary method for `CmdStanGQ` objects

Wrapper for the summary method for `CmdStanGQ` objects

## Usage

``` r
# S3 method for class 'CmdStanGQ'
summary(object, probs = c(0.05, 0.5, 0.95), ...)
```

## Arguments

- object:

  An object of class `CmdStanGQ`.

- probs:

  A numeric vector of quantiles to calculate. Defaults to c(0.05, 0.5,
  0.95).

- ...:

  Additional arguments (ignored).
