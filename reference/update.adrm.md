# Update and Re-fit a DRM Model

Update and Re-fit a DRM Model

## Usage

``` r
# S3 method for class 'adrm'
update(object, ...)
```

## Arguments

- object:

  An object of class `adrm`.

- ...:

  Arguments to be updated in the new call (e.g., .data, formula_zero,
  iter_sampling).

## Value

An updated `adrm` object.
