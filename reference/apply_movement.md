# Simple movement to population dynamics

Apply "simple" movement to population dynamics.

## Usage

``` r
apply_movement(lambda, M, mov_age)
```

## Arguments

- lambda:

  array of number of individuals per age, time, and patch

- M:

  movement matrix

- mov_age:

  ages at which movement starts (this can be generalized)

## Value

an array of numbers by age, year and patch
