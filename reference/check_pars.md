# Validate input data for [`prior_sample()`](https://pinskylab.github.io/drmr/reference/prior_sample.md)

Checks if the provided data list is valid for a given model. This is an
internal function not intended for direct use.

## Usage

``` r
check_pars(dat, model)
```

## Arguments

- dat:

  A `list` containing the data, typically generated by
  [`make_data()`](https://pinskylab.github.io/drmr/reference/make_data.md)
  or
  [`make_data_sdm()`](https://pinskylab.github.io/drmr/reference/make_data_sdm.md).

- model:

  A `character` string specifying the model type. Must be either "drm"
  or "sdm". Defaults to "drm".

## Value

This function does not return a value. It stops execution with an error
message if the data is not valid for the specified model.
