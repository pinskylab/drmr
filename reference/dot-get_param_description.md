# Internal helper to generate parameter descriptions

Internal helper to generate parameter descriptions

## Usage

``` r
.get_param_description(v, covars, model = c("drm", "sdm"))
```

## Arguments

- v:

  A character string representing the parameter name

- covars:

  A list of covariate names extracted from the model object

- model:

  A list of covariate names extracted from the model object
