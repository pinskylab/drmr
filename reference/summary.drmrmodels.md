# Summary method for `adrm` and `sdm` objects

Summary method for `adrm` and `sdm` objects

## Usage

``` r
# S3 method for class 'drmrmodels'
summary(object, probs = c(0.05, 0.5, 0.95), ...)
```

## Arguments

- object:

  An object of class `drmrmodels`.

- probs:

  A numeric vector of quantiles to calculate. Defaults to c(0.05, 0.5,
  0.95).

- ...:

  Additional arguments (ignored).
